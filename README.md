Thrive Cafe - Digital Redesign
A modern, responsive web application for Thrive Cafe, a community-centric coffee shop located in Orlando West, Soweto. This project serves as a complete digital revitalization, transitioning from a legacy web presence to a fast, sleek, and mobile-first user experience.

🌟 Project Overview
Thrive Cafe is more than a coffee shop; it is a hub for thinkers, creators, and coffee lovers in the heart of Soweto. The goal of this redesign is to reflect the cafe's vibrant energy while providing a seamless, performance-optimized digital interface for local customers to browse the menu and learn about the cafe's story.

🚀 Key Features
Modern Aesthetic: Implemented a "Stone & Earth" color palette utilizing Tailwind CSS for a premium, clean look.

Mobile-First Architecture: Fully responsive layout optimized for mobile users who visit while on the go.

Component-Driven Design: Built with reusable React components (Navbar, Menu Cards, Hero Section) to ensure scalability and maintainable code.

High Performance: Optimized with Vite for rapid load times and efficient asset management.

🛠 Tech Stack
Framework: React.js

Build Tool: Vite

Styling: Tailwind CSS

Routing: React Router
